# 範例：穿越+系統+都市重生

📖  
2026年，你在一場車禍中喪生，意識卻回到2016年的自己身上——大學剛畢業，窮得連房租都快付不出。  
腦海突然響起冰冷機械聲：

【叮！至尊重生系統綁定成功】
【宿主：林國良】
【當前任務：24小時內賺到第一桶金100萬】
【新手禮包已發放：神級投資眼光（永久）、未來30年記憶片段】

你看著手機螢幕，剛好看到比特幣價格：650美元／枚。  
宿舍門被敲響，室友大喊：「國良！房租該交了，再不交要被趕出去了！」

現在輪到你了……

你可以選擇：
1. 立刻把所有錢買比特幣，梭哈
2. 先找份兼職穩住房租，再慢慢佈局
3. 打開系統商城，看看有什麼能立刻變現的東西
4. 聯絡當年那個已經發達的同學，試著借錢或合作
